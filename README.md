# Impacto de Airbnb en el crimen en la Ciudad de México

Archivos de Jupiter que limpian las bases de datos necesarias para el análisis del impacto de Airbnb en el crimen en la Ciudad de México. Todos los datos, excepto los de Airbnb de 2019 a 2020, pueden ser solicitados.

"limpieza_crimen" tiene varias partes del código que sirven solo para visualización de datos y no se exportan. En su momento, esa visualización indicó los siguientes pasos para el análisis. Asimismo, una parte del proceso consistió en la clasificación de los crímenes en grupos. No se encontró una base pública al respecto.
